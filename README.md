
# Buenas Viagens

Projeto Acadêmico de site para uma agência de viagens desenvolvido com HTML5 e CSS3.

## Funcionalidades

- Layout responsivo
- Menu de navegação
- Seção de destinos
- Avaliações de clientes
- Footer com newsletter e redes sociais

## Tecnologias

- HTML5
- CSS3
- Font Awesome
